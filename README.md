# anonymouse
Python 3 script to send anonymous email using anonymouse.org

## Install
```
$ git clone https://github.com/thelinuxchoice/anonymouse && cd anonymouse
$ pip3 install -r requirements.txt
```

## Usage
```
$ python -V
Python 3.7.4

$ python anonymouse.py

Anonymous Email by anonymouse.org
coded by github.com/thelinuxchoice

to: toto@example.com
subject: I send you an anonymouse mail
message: Hi bro, how r u ? :D

[+] Email Sent!
[+] In order to increase your privacy, the anonymous e-mail will be randomly delayed up to 12 hours
```
